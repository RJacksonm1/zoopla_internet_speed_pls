This extension uses the Google Places Web Service API. You will need a "Server" API key for this to use this extension. This API is used to find postcodes for the partial property addresses listed in Zoopla. The postcodes are then used to fetch an internet speed estimate.

A free Google Developer Console account has a quote of 1000 requests per day for Google Places Web Service API usage. The free limit raises to 150,000 if you give Google your credit card details, but that shouldn't be necessary unless you're developing the extension. The extension caches results from the API (providing they are valid responses - 'OK' or 'ZERO_RESULTS' from the API), to reduce duplicate API calls.
