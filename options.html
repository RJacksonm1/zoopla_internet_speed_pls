This extension uses the Google Places Web Service API. You will need a "Server" API key for this to use this extension. This API is used to find postcodes for the partial property addresses listed in Zoopla. The postcodes are then used to fetch an internet speed estimate.

A free Google Developer Console account has a quote of 1000 requests per day for Google Maps API usage. The Google Places Web Service has a 10x cost, so each request to this API counts as 10 against the quota. The free limit raises to 150,000 if you give Google your credit card details.

With the free quota, you should be able to view the internet speed estimates for 100 properties (note: properties for which estimates could not be fetched will still be counted against this quota). This miiiiiiiight be enough for most people, but it wasn't for me (tho I was developing this extension).

Also the extension isn't set up to handle API requests failing due to the quota being exceeded. It will simply see that it failed to turn a partial address into a postcode (for use in getting the internet estimate), and never try to fetch a postcode for that address again.
